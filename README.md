## Overview

Swish Professional Suite is a complete gesture-based window management platform with customization depth, hotcorner integration, and advanced layout tools for professional multi-window workflows.

Custom gesture mapping assigns any action to any gesture direction. Custom gestures can trigger window positions, application launching, and system actions. Modifier keys expand gesture vocabularies.

Hot corner integration triggers window layouts when the cursor reaches screen corners. Hot corner actions can apply multi-window layouts simultaneously.

Menu bar and Dock gestures extend the vocabulary beyond the titlebar. Pinch on a Dock icon minimizes all windows of that application.

Swish Professional Suite is the window management platform for trackpad power users who want gesture-based control extending beyond basic snapping into comprehensive workspace management.

## Performance & Integration

Swish is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Swish processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
